Detailed Description

A network of neuron is what is being simulated.

An ensemble of neurons, that evolves over time and reacts to stimuli that arrive from the rest of the brain.

Constructor & Destructor Documentation

Network::Network ( )

A constructor.

Initializes a neuronal network by creating and connecting a number of neurons specified in the parameter file by means of the functions createNeurons() and establishConnections(). There is a defined ratio of inhibitory and exitatory neurons and each one of them receives a fixed number of connections from both inhibitory and excitatory presynaptic neuron that are chosen randomly.
